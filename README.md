# docker-php-fpm-boilerplate
Simple boilerplate for start local develop with NGINX, Postrgres 12, PHP 7.4, Xdebug

## Current configuration
- PHP 7.4RC06
- Xdebug
- Postgres 12
- Nginx

## Required
- docker
- docker-compose

## Run
```
git clone mikhailovv/docker-php-fpm-boilerplate projectDir
cd projectDir
docker-compose up -d
```
