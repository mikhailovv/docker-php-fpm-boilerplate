# docker-php-fpm-boilerplate
Simple boilerplate for start local develop with NGINX, Postrgres 14, PHP 8.1, Xdebug

## Current configuration
- PHP 8.1RC06
- Xdebug
- Postgres 14
- Nginx

## Required
- docker
- docker-compose

## Run
```
git clone mikhailovv/docker-php-fpm-boilerplate projectDir
cd projectDir
docker-compose up -d
```
